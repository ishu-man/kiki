Flashcards for practice!

Ready for active recall? These flashcards are automatically generated from your audio lesson. Answer honestly based on how well you recalled the information. Kiki uses your ratings and the FSRS algorithm to schedule the next review for optimal retention.

Button meanings:

Again: Select this if you could not recall the answer at all. The card will be shown again soon.
Hard: Select this if recalling the answer was possible but very difficult or took a long time..
Good: Select this if you recalled the answer correctly and with relative ease. This should be your most common choice.
Easy: Select this if you recalled the answer instantly with no effort.
